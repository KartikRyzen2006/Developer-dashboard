# 🚀 Developer Dashboard

A modern Developer Dashboard built with **HTML, CSS, and JavaScript**.

Live: https://developer-dashboard-kappa.vercel.app/

## ✨ Features

- ✅ Responsive Dashboard UI
- ✅ Task Manager
- ✅ Notes Manager
- ✅ Goals Manager
- ✅ Delete Tasks, Notes & Goals
- ✅ Local Storage Persistence
- ✅ Dark / Light Mode
- ✅ Theme Persistence
- ✅ Fetch API Integration
- ✅ API Todos
- ✅ Search Bar
- ✅ Modern Dashboard Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API
- Fetch API
- Font Awesome

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/KartikRyzen2006/Developer-dashboard.git
```

2. Open the project

```bash
cd Developer-dashboard
```

3. Open `index.html` in your browser.
